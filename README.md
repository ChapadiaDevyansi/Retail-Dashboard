# 📊 Online Retail Sales Dashboard

An interactive Business Intelligence dashboard built using **Python, Streamlit, Plotly, and Pandas** to analyze online retail sales data and generate actionable business insights.

---

## 🚀 Features

- 📈 Interactive KPI Cards
- 🌍 Country Filter
- 📅 Monthly Filter
- 💰 Revenue Analysis
- 📈 Monthly Revenue Trend
- 🌍 Top 10 Countries by Revenue
- 📦 Top 10 Products
- 🛒 Monthly Orders Analysis
- 👥 Top Customers by Revenue
- 💡 Business Insights & Recommendations

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly Express
- OpenPyXL
